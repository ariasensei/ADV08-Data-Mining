Topic 2: Variables & Data Types

🎯 What are Variables?

A variable is like a container that holds information. Just like a box can hold toys, a variable holds data.

Think of it this way:

📦 Box Name: age

📦 Box Contents: 25

The box is the variable, and what's inside is the value

📝 Creating Variables

Simple variable assignment:

name = "Alice"
age = 25
gpa = 3.8

Using variables:

print("Name:", name)
print("Age:", age)
Name: Alice
Age: 25

🏷️ Data Types in Python

Different types of information need different containers:

📝 String (str) Text

Text data enclosed in quotes

city = "Manila"
message = "Data is everywhere"
🔢 Integer (int) Whole Numbers

Whole numbers without decimals

count = 100
year = 2024
📊 Float (float) Decimal Numbers

Numbers with decimals

price = 19.99
percentage = 85.5
✓ Boolean (bool) True/False

Only two values: True or False

is_student = True
is_graduated = False
📋 List (list) Collection

Multiple values in one variable

scores = [85, 90, 78, 92]
fruits = ["apple", "banana", "cherry"]

🔍 Checking Data Types

Use type() to check what type a variable is:

print(type("Hello"))
print(type(25))
print(type(3.14))
<class 'str'>
<class 'int'>
<class 'float'>

Topic 3: Loops

🎯 What are Loops?

A loop repeats code multiple times. Imagine you need to print 100 numbers—would you write 100 print statements? No! You use a loop.

Real-world example:

📋 Process all 1000 customer records (not manually)

📋 Calculate average of 500 test scores (not one by one)

📋 Read data from 50 files (not individually)

🔄 For Loop: Repeat a Fixed Number of Times

Use a for loop when you know how many times to repeat:

Example: Print numbers 1 to 5

for i in range(1, 6):
print(i)
1
2
3
4
5

Example: Process a list of students

students = ["Alice", "Bob", "Charlie"]
for student in students:
print("Processing:", student)
Processing: Alice
Processing: Bob
Processing: Charlie

💡 How it works:

The loop takes each item from the list and assigns it to the variable (student). Then the code inside the loop runs for each item.

⏰ While Loop: Repeat Until Something Changes

Use a while loop when you want to repeat until a condition changes:

Example: Count down from 5 to 1

count = 5
while count > 0:
print(count)
count = count - 1
5
4
3
2
1

⚠️ Important:

Always make sure the condition will eventually become false, or your loop will run forever!

For vs While Loop

Aspect For Loop While Loop
When to use Known number of times Unknown, until condition changes
Example Process 100 items in list Keep asking until user says stop
Data Mining Use 🌟 More Common Less Common

Topic 4: Functions

🎯 What are Functions?

A function is reusable code. Instead of writing the same code over and over, write it once in a function and use it many times.

Real-world analogy:

🏠 Instead of describing how to make coffee every time someone visits, you just say "I'll make coffee" and use the same process.

Functions are like recipes—define once, use many times.

📝 Creating a Function

Simple function with no input:

def greet():
print("Hello, Data Miners!")
# Call the function
greet()
Hello, Data Miners!

Function with input (parameters):

def add(a, b):
return a + b
# Call the function
result = add(5, 3)
print(result)
8

🔧 Parts of a Function

def calculate_average(scores):
# This is the function body
total = sum(scores)
count = len(scores)
average = total / count
return average

def keyword

Tells Python you're defining a function

Function name

What you call it: calculate_average

Parameters (input)

Data the function receives: scores

Function body

Code that runs inside the function (indented)

Return (output)

What the function gives back

💡 Practical Data Mining Example

Function to normalize test scores (0-100):

def normalize_score(score, max_score):
normalized = (score / max_score) * 100
return normalized
# Using the function
score1 = normalize_score(45, 50)
score2 = normalize_score(85, 100)
print("Score 1:", score1)
print("Score 2:", score2)
Score 1: 90.0
Score 2: 85.0

Topic 5: NumPy & Pandas - The Data Mining Superpowers

🎯 What are NumPy and Pandas?

NumPy and Pandas are Python libraries that make working with data super easy. They're the tools every data scientist uses.

🔢 NumPy: Fast math operations on arrays

📊 Pandas: Work with tables (like Excel spreadsheets)

⚡ Both are lightning-fast compared to regular Python

🔢 NumPy - Numerical Computing

NumPy helps with mathematical calculations on large arrays:

Import and use NumPy:

import numpy as np
# Create an array
scores = np.array([85, 90, 78, 92])
# Calculate statistics
print("Mean:", np.mean(scores))
print("Max:", np.max(scores))
print("Min:", np.min(scores))
Mean: 86.25
Max: 92
Min: 78

✅ NumPy is great for:

  • • Mathematical calculations
  • • Working with numerical data
  • • Matrix operations
  • • Statistical functions

📊 Pandas - Data Analysis

Pandas works with DataFrames (tables with rows and columns):

Import and create a DataFrame:

import pandas as pd
# Create a table
data = {'Name': ['Alice', 'Bob'],
'Score': [85, 90]}
df = pd.DataFrame(data)
print(df)
  Name Score
0 Alice     85
1  Bob     90

✅ Pandas is great for:

  • • Reading Excel files
  • • Cleaning data
  • • Organizing tables
  • • Data analysis & filtering

Real-World Case: Analyzing Student Grades with Python

The Problem

A university has grades for 1,000 students across 50 courses. How do you find the top performers, average grades, and struggling students? Manually? That would take forever!

The Python Solution

Using Pandas to analyze grades:

import pandas as pd
# Load the data
grades = pd.read_csv('student_grades.csv')
# Calculate average grade
average = grades['Score'].mean()
print("Average grade:", average)
# Find top 10 students
top_students = grades.nlargest(10, 'Score')
# Find students below 70 (need help)
struggling = grades[grades['Score'] < 70]

💡 The Impact:

Instead of hours of work, Python delivers results in seconds! The university can quickly identify struggling students and provide help.
